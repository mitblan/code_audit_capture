## Data Security

This repository excludes proprietary and company-sensitive data.

Do not commit:
- RVIA databases or exports
- Company write-up data
- Generated CSV or PDF files
- Local SQLite databases

Use the following directories for local-only data:
- `assets/private_data/`
- `imports/`
- `exports/`
- `local_data/`

Safe sample data for development may be stored in:
- `assets/sample_data/`
